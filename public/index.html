This is a simple guide on how the api works
  1. To get all users :
    Send a get request to link/users
    GET
    link/users
  2. To add user :
    Send a post request with user data in the body minus the id as it is auto generated and assigned in the server
    POST
    link/users


  3. To get data on specific user :
    Send a get request to link/users/userid
    GET
    link/users/userid
  4. To patch user details :
    Send a patch request to link/users/userid with the data to be changed in the body using postman
    PATCH
    link/users/userid
  5. To delete user :
    Send a delete request to link/users/userid using post man
    DELETE
    link/users/userid
